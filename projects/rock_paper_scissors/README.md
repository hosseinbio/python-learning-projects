# Rock, Paper, Scissors Game (Python)

A simple **Rock, Paper, Scissors** game implemented in Python where a user plays against the computer.  
The game keeps track of scores, shows the winner of each round, and allows you to quit anytime by pressing **'q'**.

---

## Features
- Interactive text-based gameplay
- Random computer choices
- Real-time score tracking
- Option to quit at any time
- Simple, clean, and beginner-friendly Python code

---

## How the Game Works
1. The program asks you to choose between:
   - `rock`
   - `paper`
   - `scissors`
2. The computer randomly selects one of the three options.
3. The winner of each round is determined by standard rules:
   - Rock beats Scissors  
   - Scissors beats Paper  
   - Paper beats Rock  
   - Same choice = Tie
4. The scores are updated and displayed after each round.
5. You can quit the game anytime by pressing **'q'**.

